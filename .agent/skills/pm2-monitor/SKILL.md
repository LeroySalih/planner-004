# PM2 Health Monitor Skill

## Description

Use this skill to inspect running PM2 processes, check memory usage, and
identify if any services have crashed.

## Capabilities

- List all active PM2 processes.
- Check system resource usage (CPU/Memory).
- Restart specific processes if they show high error counts.

## Usage Instructions

Load this skill when the user asks "How are my apps doing?" or "Check system
health."
